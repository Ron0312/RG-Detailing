## 2025-03-14 - Improve Keyboard Accessibility Focus Rings
**Learning:** Adding `type="button"` and `focus-visible` utility classes systematically to custom interactive elements (like custom toggles or mobile menu hamburger icons) greatly enhances keyboard navigation predictability, particularly on mobile and custom components. Native `<button>` elements without explicit types inside forms can unintentionally trigger submissions.
**Action:** When creating any custom interactive element, explicitly specify `type="button"` and include visible focus rings (`focus-visible:ring-2 focus-visible:ring-red-500 focus-visible:outline-none focus-visible:ring-offset-2 focus-visible:ring-offset-zinc-900`) using Tailwind to ensure consistent keyboard accessibility across the application.
